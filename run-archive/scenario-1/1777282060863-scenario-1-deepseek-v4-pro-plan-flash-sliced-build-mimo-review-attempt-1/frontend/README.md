# Restaurant Booking — Frontend

React SPA for the restaurant booking system.

## Scripts

```bash
npm run dev           # Start Vite dev server on :5173
npm run build         # tsc -b && vite build
npm run typecheck     # tsc --noEmit
npm run lint          # ESLint
npm run format:check  # Prettier check
npm run deadcode      # Knip dead code analysis
npm run generate-api  # Regenerate typed client from backend OpenAPI
```

## Generated API Client

The typed API client lives in `src/api/` and is generated by Orval from `openapi.json`. To regenerate:

1. Start the backend on `http://localhost:5000`
2. Run `npm run generate-api`

All data fetching uses these typed functions — there are no hand-written fetch wrappers.
